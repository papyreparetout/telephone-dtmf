# telephone-dtmf
transforme un téléphone à cadran (numérotation à impulsion) en téléphone DTMF

A l'aide d'un arduino et d'un circuit d'interface à la ligne téléphonique on peut transformer un ancien téléphone à cadran U43 ou S63
pour que la numérotation se fasse en utilisant des signaux DTMF

Ce la permet d'interfacer un ancien téléphone avec une box
